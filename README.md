# A sample mobile application built with React Native and GraphQL.

### Features

- Show list of Users

## Tech Stack

- JavaScript
- React
- React Native
- Apollo Client for manage remote data

## Build Setup

- Install dependencies

```bash
# (Yarn)
$ yarn
# (npm)
$ npm install
```

- Run

```bash
 $ npx react-native run-android
```
